#  Barbearia

## Sobre o Projeto

Este é um projeto de **sistema de barbearia** desenvolvido para gerenciar funcionalidades relacionadas a uma barbearia — como cadastro de serviços, agendamentos, clientes e fluxo básico de operação. Ele foi criado com foco em aprender e demonstrar habilidades práticas no desenvolvimento de aplicações completas em um contexto real. :contentReference[oaicite:1]{index=1}
